#Repositório de Tralahos da disciplina de Inteligência Artificial UFMA


Atividade 01 - ## Descrição do Agente: Algoritmo A* no Sistema Solar

O agente representado pelo algoritmo A* atua como um solucionador de problemas de busca em grafos. Seu objetivo é 
encontrar o caminho mais curto entre astros do Sistema Solar, considerando as distâncias entre eles. O agente é 
classificado como:
- **Racional**: escolhe ações que otimizam o custo até o destino;
- **Determinístico**: as transições no grafo são fixas e conhecidas;
- **Solucionador de Problemas**: pertence à categoria de agentes que resolvem problemas de busca.

Atividade 03 - ## Análise de dados e classificação no dataset SolarFlare

O dataset Solar Flare é utilizado para classificação multioutput.

O objetivo é, a partir das 10 características preditoras (entradas), classificar em uma das três classes
de erupções solares são previstas, representadas nas três últimas colunas:
• Produção de erupções da classe C por esta região nas próximas 24 horas (erupções comuns)
• Produção de erupções da classe M por esta região nas próximas 24 horas (erupções moderadas)
• Produção de erupções da classe X por esta região nas próximas 24 horas (erupções severas)
